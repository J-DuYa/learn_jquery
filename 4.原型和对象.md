#### 构造函数模式

###### 创建一个jQuery对象（类）
```javascript
//每个jQuery对象包含两个属性name,selector和一个方法show
function jQuery(selector) {
	this.name = 'jQuery';
	this.selector = selector;
	this.show = function () {
		console.log(this.selector);
	};
}

//通过new关键字创建3个实例
var a = new jQuery('a');
var b = new jQuery('b');
var c = new jQuery('c');
```
* 每个实例(a, b, c)都包含属性name,selector，方法show
* show对于每个实例都有一份拷贝，各自占用内存空间。

###### 使用原型
```javascript
function jQuery(selector) {
	this.name = 'jQuery';
	this.selector = selector;
}

jQuery.prototype = {
	constructor: jQuery,
	show: function () {
		console.log(this.selector);
	}
};

var a = new jQuery('a');
var b = new jQuery('b');
var c = new jQuery('c');
```
* 每个实例(a, b, c)的show方法都指向同一个引用，只占用一份内存空间。
* 需要通过scope连接到原型链上查找，这样就无形之中要多一层作用域链的查找了
